# Global Heat-Flow Anomalies and Tectonic Distance Analysis
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18725719.svg)](https://doi.org/10.5281/zenodo.18725719)

This repository contains the full reproducible analysis pipeline used in the study:

## Persistence of intraplate heat-flow anomalies far from tectonic boundaries

The project investigates the spatial relationship between terrestrial heat-flow measurements and tectonic structures, with a focus on intraplate anomalies that remain unexplained by classical conductive and tectonic models.

The analysis shows that elevated heat-flow values persist at large distances from tectonic boundaries and remain statistically robust under multiple spatial and sampling controls, suggesting the presence of a residual intraplate thermal component not explained by classical tectonic or conductive mechanisms.

All results reported in the manuscript are fully reproducible from the pipeline, and all figures correspond to versioned outputs included in this repository.
